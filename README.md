# Gantt Chart Progress & Report

A vanilla JavaScript Gantt chart application with drag-to-move/resize task bars, localStorage persistence, and report generation utilities. Runs directly in the browser via `index.html` and includes Jest tests for core logic.

## Getting started

1. Install dependencies (Jest for tests):

```bash
npm install
```

2. Open the app by loading `index.html` in a modern browser.
3. Run the unit tests:

```bash
npm test
```

## Features

- Local storage persistence with sample seed data.
- Interactive Gantt chart with draggable and resizable task bars.
- Quick progress sliders, ad-hoc task logging, and work log capture.
- Text report generation that can include ad-hoc tasks and work logs.
- Jest tests covering persistence, date helpers, and report generation.
