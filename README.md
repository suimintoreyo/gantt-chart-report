# ガントチャートレポート

ブラウザだけで動くガントチャート型の進捗管理・レポートアプリを作るためのリポジトリです。タスクバーのドラッグ移動やリサイズ、ローカル保存、進捗報告文の生成などを想定しています。

## 目的と機能（想定）
- プロジェクト/タスクをガントチャートで視覚管理
- バーのドラッグ/リサイズで開始日・終了日を編集
- ローカルストレージでデータを永続化
- 進捗報告文を自動生成
- 主要ロジックは Jest でユニットテスト予定

## 動作環境
- フロントエンドのみ（HTML/CSS/JavaScript、フレームワークなし）
- サーバー不要で最新デスクトップブラウザ上で動作予定
- データは localStorage に保存

## 開発・テストのメモ
- エントリーポイント: `index.html`（予定）
- テスト: Node + Jest を使用予定。環境構築後に `npm test` を実行。
