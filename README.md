# ガントチャート進捗・報告アプリ

## 概要
ガントチャートベースの進捗管理・報告アプリの要件検討用リポジトリです。ブラウザのみで動作するシンプルなUIを想定し、ドラッグ＆ドロップによるタスク編集やローカル永続化などの機能を計画しています。

## ブランチ運用
- 基本的に現在の作業ブランチ上で変更を積み上げ、不要な派生ブランチは作成しません。
- レビュー対応などで作業ブランチを明示する必要がある場合は、`claude/review` 系のブランチ名を使用します。
- 必要に応じて作業内容を明示したコミットをまとめ、同ブランチからのPRを前提とします。

### CodexによるPRレビュー
- 現在の作業ブランチから `dev_claude` へ出したプルリクは、PRリンク（例: GitHubのPR URL）を共有すれば Codex がレビュー可能です。
- 気になる観点や確認してほしい項目があれば、PRリンクと併せて明示してください。

主な検討項目:
- タスクのドラッグ＆ドロップ編集と期間調整
- 進捗率・担当者などメタ情報の入力と可視化
- ブラウザローカルへの永続化（localStorage想定）
- 日付範囲を指定した進捗・課題レポート生成
